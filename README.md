Planilha automática usada por jogadores do RPG Jujutsu Kaisen Reborn.

A planilha é atualizada semanalmente, trazendo novas caçadas aleatorizadas selecionadas dentro de uma lista em uma aba secundária, as caçadas são aleatorizadas com padrões de recompensa pré-definidos.
